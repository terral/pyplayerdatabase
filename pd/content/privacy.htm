Privacy Policy

Last updated: December 4, 2012

This is currently beta software. We value privacy and security, but use at your own risk.

Some web server logs are stored for security purposes, and include IP addresses and requested URLs. IP address information is not used for any other purpose.

Cookies are used exclusively to store session information for logged-in users.

Passwords as stored in the database are encrypted using PBKDF2. That said, please still use good password practices and don't re-use passwords with this site.

Any information, comments, names, etc. submitted to this web site, unless otherwise stated, are publicly-accessible.

We assume no responsibilities for content contained in web sites linked to from this one.
